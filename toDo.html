To-Do List

The goal of this project was to create a simple to-do list with the ability to create high-priority tasks and drag them to change the order in which the tasks were in.

Languages HTML, CSS, and JavaScript.

Tools

Visual Studio Code for development and Git/GitHub.

Concepts

DOM Manipulation: I was introduced to DOM Manipulation and how to implement it to create new customizable tasks with interactability. Class Uses: I gained a more in-depth understanding of how to use classes to ensure consistency when creating new elements. Local Storage: I learned how to implement local storage in order to provide a list of tasks that would stay for users who may require them later throughout their day or later in the week.

Challenges & Solutions

The Solution: Throughout class I would ask those who had already made considerable progress or were finished with their local storage and learn from their logic and how they implemented it within their own projects. The Problem: I had difficulty learning the basics of creating new elements using Javascript and how to place them within other elements. The Solution: In order to handle this I was able to pull knowledge from my peers as well as my AP Computer science class from my home school which showed me a simpler version of the issue providing me a solid understanding to build on The Problem: I was unsure exactly what JSON had to do with local storage or what its purpose was when dealing with local storage. The Solution: I worked on my code and when it wouldn’t work I would reach out to others and see them using JSON. I then looked through resources our teacher gave us and did general research on its uses.
